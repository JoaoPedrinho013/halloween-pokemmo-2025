<h2>Funcionalidades do Site</h2>

<h3>1. Página Inicial (Desktop)</h3>
<ul>
  <li>Logo temático com sprite do Pumpking</li>
  <li>Título estilizado "Halloween PokeMMO 2025"</li>
  <li>Subtítulo descritivo</li>
  <li>Botão para download oficial do PokeMMO com efeitos de hover</li>
  <li>Links para GitHub, LinkedIn, WhatsApp e Fórum do Evento</li>
</ul>

<h3>2. Tabela Completa do Farm (Desktop)</h3>
<ul>
  <li>Tabela exibindo todos os itens farmados:</li>
  <ul>
    <li>Pumpkin Goodie Bag</li>
    <li>Mysterious Dusk Ball</li>
    <li>Special Candy</li>
    <li>Spooky Candy</li>
    <li>Goodie Bag</li>
  </ul>
  <li>Exibição de total, média, mínimo, máximo e total vendido</li>
  <li>Linha especial de TOTAL GERAL</li>
  <li>Slide adicional com informações gerais do farm</li>
</ul>
